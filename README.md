# Attendance Management System

## Description
A simple console-based Attendance Management System built using Java. 
It allows users to add students, mark attendance, and view attendance records.

## Features
- Add students
- Mark attendance (Present / Absent)
- View attendance history

## Technologies Used
- Java
- Collections (HashMap & ArrayList)
- Console-based interaction

## How to Run
1. Clone the repository
2. Compile and run:
   ```bash
   javac Attendance.java
   java Attendance
